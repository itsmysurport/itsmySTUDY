## 명령모드

- vim에 직접 명령을 내리는 모드
- 커서 이동
  - **위로 움직임** : k
  - **아래로 움직임** : j
  - **왼쪽으로 움직임** : h
  - **오른쪽으로 움직임** : l
  - **다음 단어의 끝으로 이동** : e
  - **다음 단어의 처음으로 이동** : w
  - **현재 줄 위의 맨앞으로 이동**  : -
  - **현재 줄 맨 앞으로 이동** : ^
  - **현재 줄 맨 뒤로 이동** : $
  - **문장의 맨 마지막으로 이동** : Shift + g
  - **n의 라인으로 이동 : ngg**
- 화면 스크롤
  - **한 화면을 앞으로 스크롤** : Ctrl + F
  - **한 화면을 뒤로 스크롤** : Ctrl + B
  - 반 화면을 앞으로 스크롤 : Ctrl + D
  - 반 화면을 뒤로 스크롤 : Ctrl + U
  - **한줄 앞으로 스크롤** : Ctrl + E
  - **한줄 뒤로 스크롤** : Ctrl + Y
- 입력 명령
  - a, i, o, s
  - A, I, O, S

- 절편집
  - 한줄 복사 : y, yy
  - 현재 라인으로 부터 n라인 만큼 복사 : yn, nyy
  - 한줄 삭제 : dd
  - 현재 커서로 부터 마지막까지 삭제 : Shift + d
  - 현재 행의 개행문제 제거 : shift + j
- Undo : u
- 절 찾기 / 변환
  - **문서의 처음부터 마지막까지의 n을 _n_으로 취환하다** : *:%s/n/_&_/g*
